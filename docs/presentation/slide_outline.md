# 발표 자료 — 슬라이드 아웃라인

> 목적: 기획서의 핵심 메시지를 _발표 흐름_ 으로 재배열한 청사진. 슬라이드 제작 도구(Keynote / PPT / Pitch / Gamma 등)에서 실제 슬라이드를 만들기 전 단계.

> 권장 분량: 12~15장 / 발표 시간 7~10분 기준.

## 발표 흐름 (Narrative Arc)

> 기획서는 논리 순이지만, 발표는 _감정 곡선_ 을 따라가는 것이 효과적이다.

1. **Hook** (1~3p) — 청중이 _자기 일_ 처럼 느낄 한 장면
2. **Pain** (4~6p) — 문제의 크기·심각성
3. **Solution** (7~12p) — 우리의 답
4. **Impact** (13~14p) — 그래서 세상이 어떻게 바뀌는가
5. **Close** (15p) — 한 줄 + 팀

---

## 발표 디자인 가이드

- **글자 크기**: 본문 ≥ 24pt, 제목 ≥ 36pt
- **컬러**: 메인 1색 + 강조 1색 (3색 이상 금지)
- **이미지·다이어그램**: 텍스트보다 우선
- **폰트**: Pretendard / Spoqa Han Sans / Noto Sans KR 권장
- **장당 메시지**: 1슬라이드 = 1메시지
