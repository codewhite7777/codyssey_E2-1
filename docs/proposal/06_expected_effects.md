# 6. 기대 효과

기대 효과는 가능한 한 수치로 표현하되, 실측 전 수치는 가설임을 명시한다.  
본 서비스의 핵심 가치는 숏폼 광고를 본 뒤 발생하는 검증 시간·충동 구매 위험·허위 광고 노출 피해를  
사용자 개입 최소화로 줄이는 데 있다.

---

## 6.1 사용자 관점

**편익 1 — 광고 검증 시간 단축**

기존 방식(댓글 확인 + 네이버 검색 + 후기 비교)은 평균 5~10분이 소요된다.  
서비스 도입 후 공유 1회 + 결과 확인으로 60초 이내 검증이 가능하다.  
_목표: 검증 시간 80% 이상 단축 (파일럿 실측 후 보정 예정)_

**편익 2 — 충동 구매 억제**

광고 직후 별도 검증 없이 구매하는 패턴에서, 고위험 광고에 대해 구매 보류 또는 추가 검증 행동을 유도한다.  
_목표: 고위험 광고 즉시 구매율 30% 이상 감소 (파일럿 실측 후 보정 예정)_

**편익 3 — 가족 검증 부담 감소**

부모님이 보낸 광고를 자녀가 대신 검색·설명하는 데 1건당 5~15분이 걸린다.  
분석 결과를 공유하면 1~2분 내 설명이 가능하다.  
_목표: 가족 검증 시간 70% 이상 절감 (파일럿 실측 후 보정 예정)_

---

## 6.2 비즈니스 관점

**셀러·플랫폼 관점 가치**

AI 1차 분석을 활용하면 광고 100건 수작업 검토(8~10시간)를 1~2시간 수준으로 줄일 수 있다.  
허위·과장 광고 노출 감소로 사용자 신고·민원 대응량도 줄어들 수 있다.

**수익 모델**

| 플랜 | 가격 (가설) |
|------|------------|
| 개인 프리미엄 | 월 4,900~6,900원 |
| 가족 플랜 | 월 9,900~12,900원 |
| 기관·플랫폼용 대시보드/API | 월 50만~300만 원 (구독 또는 계약형) |

_유료 전환율 목표: 1차 3~5% / 6개월 7~10%_

**시장 진입 방향**

2025년 9월 기준 월간 온라인쇼핑 거래액은 약 23조 8,000억 원이며, 모바일쇼핑 비중은 75.9%다.[^1]  
초기에는 건강기능식품·화장품·다이어트 광고 검증으로 좁게 진입하고,  
핵심 타깃 내 초기 활성 사용자 1,000명 확보를 1차 목표로 설정한다.

---

## 6.3 사회·산업 관점

**정보 비대칭 해소**

식약처의 2025년 숏폼·라이브커머스 점검에서 식품 광고 65%(225건 중 147건), 화장품 광고 73%(100건 중 73건)가 표시광고법 위반으로 판정됐다.[^2]  
본 서비스는 이처럼 높은 위반율 환경에서 사용자가 별도 검색 없이도 위험 신호를 인지할 수 있게 한다.

**취약 사용자 보호**

중장년·고령층·디지털 취약층이 AI 합성 가짜 전문가 광고에 바로 설득될 가능성을 낮춘다.  
2025년 한 해 AI 생성 '가짜 전문가' 활용 부당광고 63건이 적발된 상황에서,[^3] 수요는 실재한다.

**기관·플랫폼 보조 효과**

장기적으로 광고 모니터링 보조 도구로 활용 가능하며, 수작업 모니터링 대상 선별 효율을 높일 수 있다.

## 6.4 정량 KPI

| KPI | 정의 | 1차 목표 | 6개월 목표 |
|-----|------|----------|------------|
| 평균 분석 완료 시간 | 광고 공유 후 리포트 도착까지 평균 시간 | 60초 이내 | 30초 이내 |
| 고위험 광고 탐지 정밀도 | 위험 판정 중 실제 위험 광고 비율 (효능 과장·**딥페이크 포함**) | 85% | 90% |
| 딥페이크 판별 AUC | F4 모듈의 AI 합성 인물 탐지 AUC-ROC | 0.85 | 0.90 |
| 고위험 광고 즉시 구매 억제율 | 고위험 광고에서 즉시 구매를 멈춘 비율 | 30% | 50% |
| 사용자 재사용률 | 첫 사용 후 30일 내 2회 이상 재사용 비율 | 25% | 40% |
| 결과 이해도 만족도 | "결과가 이해하기 쉬웠다" 응답 비율 | 75% | 85% |
| 가족 공유율 | 결과를 가족·지인에게 공유한 비율 | 10% | 20% |
| 유료 전환율 | 무료 사용자 중 유료 결제 전환 비율 | 3% | 7% |
| 누적 분석 광고 건수 | 서비스 시작 후 총 분석 완료 건수 | 300건 | 2,000건 |

> KPI 수치 중 사용자 행동 관련 항목(억제율·공유율 등)은 파일럿 실측 후 보정한다.

## 6.5 잠재적 부작용·고려사항

| 위험·부작용 | 발생 시나리오 | 대응 |
|-------------|---------------|------|
| 오탐 (False Positive) | 문제 없는 광고를 위험으로 표시 | 위험도를 3단계 이상으로 나누고 "주의"와 "고위험"을 분리 |
| 미탐 (False Negative) | 실제 위험 광고를 놓침 | 사용자 피드백 반영, 월 1회 이상 룰셋·모델 보정 |
| 과도한 의존 | 사용자가 결과만 믿고 스스로 판단하지 않음 | "판단 보조 도구"임을 명시하고 근거 문구를 함께 제시 |
| 규정 반영 지연 | 정책 변경 후 기준이 늦게 반영됨 | 정책 DB 주 1회 이상 갱신, 분기별 규정 점검 |
| 프라이버시 우려 | 공유한 광고 데이터 저장에 부담 | 최소 수집, 30일 이내 원본 삭제, 익명화 저장 |
| 처리 비용 증가 | 사용량 증가 시 모델 호출 비용 상승 | 경량 모델 우선, 고난도 샘플만 고성능 모델 fallback |
