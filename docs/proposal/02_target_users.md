# 2. 타겟 사용자

본 서비스가 해결하려는 문제는 숏폼 광고를 스쳐 지나가는 짧은 순간 안에서  
건강기능식품·화장품·다이어트 상품의 허위·과장 여부를 실시간으로 판단하기 어렵다는 데 있다.  
특히 이 문제는 영상 기반 정보 소비가 높은 중장년·고령층에서 더 크게 나타나며,  
권위 연출·후기 형식·극적인 전후 비교와 같은 광고 장치에 취약한 사용자에게 더 직접적인 피해로 이어진다.

따라서 본 서비스는 규제 민감 카테고리의 광고에 반복적으로 노출되고,  
실제 구매 의사결정을 수행하며, 광고를 의심해도 즉시 검증할 도구가 없는 사용자를 1차 타깃으로 설정한다.

## 2.3 사용자 니즈 분석

### 정량 — 시장 규모 및 문제 빈도

**이커머스·모바일쇼핑 시장**

2025년 9월 기준 월간 온라인쇼핑 거래액은 약 23조 8,000억 원이며, 이 중 모바일쇼핑 비중은 75.9%에 달한다.[^2] 국내 온라인쇼핑은 해마다 성장세를 유지하고 있으며, 건강기능식품·화장품 카테고리는 숏폼 기반 구매 전환이 활발한 대표 카테고리다.

**숏폼·영상 기반 정보 소비**

2025년 언론수용자 조사에 따르면, 온라인 동영상 플랫폼을 통한 뉴스·정보 이용률이 전년 대비 11.6%p 급증했으며 해당 플랫폼 중 유튜브가 92.2%의 점유율로 사실상 독점적 지위를 차지한다.[^3] 숏폼 뉴스 이용률 역시 1년 새 11.1%에서 22.9%로 두 배 이상 뛰었다.[^3]

**부당광고 적발 현황**

식약처의 2025년 숏폼·라이브커머스 기획 점검 결과, 부당광고 적발 건수는 식품 298건·화장품 83건으로 집계됐으며, 주된 위반 유형은 의약적 효능·효과 표방 광고였다.[^4] 동년 AI로 생성한 '가짜 전문가'를 활용한 부당광고만 63건이 별도 적발되었다.[^4] 별도 집중 점검에서는 SNS 내 숏폼 식품 광고 225건 중 147건(65%), 화장품 광고 100건 중 73건(73%)이 식품표시광고법 위반으로 판단됐다.[^5]

**TAM / SAM / SOM 추정**

| 구분 | 정의 | 규모 추정 |
|------|------|----------|
| TAM | 모바일 숏폼을 이용하는 전체 소비자 | 전체 온라인쇼핑 이용자 수준 |
| SAM | 건강·뷰티·다이어트 상품에 관심이 있고 숏폼 광고를 통해 상품을 접하는 40~60대 여성 | 해당 연령대 모바일 쇼핑 이용자 |
| SOM | 수도권·모바일 사용 익숙층·공유 기능 사용 가능한 50~60대 초반 여성 | 초기 파일럿 기준 100~500명 |

### 정성 — 핵심 인사이트

- 정보 문제: 광고·후기·협찬·전문가 정보의 경계가 모호하다.
- 시간 문제: 의심이 들어도 바로 검증하기 번거롭다.
- 신뢰 문제: 의사·전문의·교수·실사용 후기 형식을 신뢰하기 쉽다.
- 감정 문제: 건강·노화·체형·피부 고민은 불안감을 자극해 판단을 흐리게 만든다.

**사용자 니즈 우선순위**

1순위: 광고가 왜 의심스러운지 빠르게 알고 싶다.  
2순위: 어려운 법률 용어 말고 쉬운 설명을 원한다.  
3순위: 가족에게 보여줄 수 있는 형태의 결과를 원한다.  
4순위: 공유 한 번으로 끝나길 원한다.

## 2.4 사용자 시나리오

### As-Is (현재)

정희는 저녁 식사 후 유튜브 Shorts를 본다. 영상 하나에서 "전문의가 추천하는 관절 영양제", "2주 만에 붓기 개선", "실제 사용자 후기" 같은 문구가 나온다. 영상 속 인물은 흰 가운을 입고 말투도 전문가처럼 느껴진다.

더 알아보고 싶지만 지금 검색하기가 번거롭다. 댓글은 광고성으로 보이고, 링크를 눌러도 할인 문구와 후기만 가득하다. 다시 검색하면 광고와 정보가 섞여 무엇을 믿어야 할지 헷갈린다. 결국 찜만 해두거나, 할인 종료가 두려워 결제하거나, 찜찜한 상태로 그냥 넘긴다.

영수 씨의 경우는 더 직접적이다. 비슷한 상황에서 AI로 합성된 가짜 의사가 등장한 알부민 광고를 접하고, 일반 식품인 난백 알부민을 간 기능 개선제로 믿어 수십만 원에 구매한 사례가 실제로 보고되어 있다.[^1]

### To-Be (서비스 도입 후)

정희는 같은 상황에서 영상이 수상하다고 느껴지면 공유 버튼을 눌러 앱을 호출한다. 서비스는 광고의 영상·자막·주장·링크를 빠르게 분석한 뒤 리포트를 보여준다.

리포트에는 다음 내용이 정리된다.

- 전문의·교수형 권위 연출 사용
- 효능을 단정적으로 표현한 과장 가능성 높은 문구
- 전후 비교 중심의 시각 연출
- 주의가 필요한 이유와 관련 근거

정희는 더 이상 광고를 감으로만 판단하지 않는다. 앱이 대신 결정해주는 게 아니라, "왜 조심해야 하는지"를 빠르게 이해하게 된다. 그 결과 충동 구매를 줄이고, 진짜 검증이 필요한 상품과 무시해도 되는 광고를 구분할 수 있게 된다.

서연의 경우, 부모님이 보낸 광고 링크를 받아 같은 방식으로 분석해볼 수 있다. 반복적인 검색 시간을 줄이고, 부모님을 설득할 때 객관적인 근거를 손쉽게 제공할 수 있다.

## 2.5 비타겟 명시

서비스 정체성을 분명히 하기 위해, 다음 사용자는 이번 서비스의 1차 고객으로 보지 않는다.

- 광고주·마케터: 본 서비스는 광고 성과 최적화 도구가 아니라 소비자 판단 보호 도구다.
- 플랫폼 내부 심사 운영자: 장기적으로 확장 가능성이 있으나, MVP 단계의 직접 공략 대상은 아니다.
- B2B 구매 담당자: 산업재·도매·기업 구매는 본 서비스가 다루는 숏폼 기반 감성·권위형 광고 문제와 결이 다르다.
- 최저가 비교 사용자: 본 서비스의 본질은 가격 비교가 아니라 허위·과장 광고 판별이다.
- 완전 자동 감시를 기대하는 사용자: MVP는 피드 위 자동 오버레이가 아니라, 사용자가 의심한 순간 공유 시트로 호출하는 능동형 구조다. 플랫폼·OS 제약을 고려한 현실적인 선택이다.
