# 2. 타겟 사용자

본 페르소나는 1번 파일의 데이터와 최근 식통계 약처·공정위가 발표한 실제 부당 광고 적발 사례(알부민, AI 가짜 의사 등)를 근거로 설계되었다.

-----

## 2.1 주 타겟 페르소나

| 항목 | 내용 |
|------|------|
| **이름·연령·직업** | **김지혜 (32세) / 직장인** |
| **거주·환경** | 서울 거주 1인 가구. 퇴근 후 침대에서 스마트폰 숏폼(릴스, 쇼츠)을 보는 것이 유일한 휴식 루틴. |
| **이커머스 사용 빈도·채널** | 일 평균 90분 이상 영상 시청. 인스타그램 및 유튜브 피드 광고를 통해 주 2\~3회 충동구매. |
| **페인포인트 (Top 3)** | 1) **비판적 사고 마비**: 숏폼의 빠른 흐름 속에서 광고의 진위를 검색해 볼 여유가 없음.<br>2) **네이티브 광고의 기만**: 일반 리뷰와 광고의 경계가 모호해 속았다는 불쾌감을 자주 느낌.<br>3) **정보 과부하**: '알부민', '다이어트 보조제' 등 유사 광고가 쏟아져 실제 인증 제품을 혼동함. |
| **현재 대처 방식** | 광고 댓글을 확인하거나(주로 조작됨), 결제 직전 포털에서 상품명을 재검색함. |
| **기대하는 것** | 의심되는 순간 공유 시트를 통해 3초 내로 광고의 규제 위반 여부나 AI 생성 여부를 즉시 판별해 주는 '비판적 조력자'. |

## 2.2 부 타겟 페르소나

| 항목 | 내용 |
|------|------|
| **이름·연령·직업** | **박영수 (68세) / 은퇴자 (유튜브 헤비 유저)** |
| **페인포인트** | **가짜 권위에 대한 무조건적 신뢰**: 흰 가운을 입은 인물(AI 합성 의사)이 "간 질환에 알부민이 특효"라고 말하면 이를 공신력 있는 정보로 믿음. 일반 식품(난백)과 의약품의 차이를 인지하지 못해 고가에 구매함. |
| **우선순위** | 고령층의 높은 유튜브 뉴스 신뢰도(93.1%)와 디지털 취약성으로 인해 **가장 심각한 경제적 피해**를 입는 계층임. 직접 앱을 설치하기보다 자녀가 '안전 장치'로서 설정해 줄 가능성이 높음. |

  - **사례 기반 도출**: '먹는 알부민' 광고처럼 의사 사칭 AI 영상은 전문가에 대한 신뢰를 악용함.

## 2.4 사용자 시나리오 (실제 사례 반영)

### As-Is (현재)

1.  박영수 씨가 유튜브에서 "간 기능 개선 특효, 알부민" 광고를 접함.
2.  AI로 합성된 가짜 의사가 등장해 "이걸 안 먹으면 간 수치가 위험하다"고 경고함.
3.  영수 씨는 전화를 걸어, 단순 달걀 추출물(난백 알부민)인 일반 식품을 수십만 원에 구매함.
4.  나중에야 이것이 치료제가 아님을 알고 자책함. ([참조: 식약처 '알부민 식품' 부당광고 적발 사례](https://www.daejeonpress.co.kr/news/68801))

### To-Be (서비스 도입 후)

1.  영수 씨가 의심스러운 알부민 광고를 보던 중 **[광고 검증]** 버튼을 누름.
2.  서비스 AI가 영상 내 '간 질환 치료' 표현이 식약처 규제 위반임을 3초 내로 분석함.
3.  화면에 **"⚠️ 주의: 이 제품은 일반 식품입니다. 의사 사칭 의심"** 경고 문구가 노출됨. ([참조: 공정위 AI 가상인물 표시 의무화 지침](https://www.ftc.go.kr/www/selectBbsNttView.do?pageUnit=10&pageIndex=1&searchCnd=all&key=12&bordCd=3&searchCtgry=01,02&nttSn=47317))
4.  영수 씨는 구매를 포기하고 사기 피해를 예방함.

## 2.5 비-타겟 명시

  - **C2C 중고 거래자**: 당근마켓 등 개인 간의 사기 행위는 분석 범위에서 제외한다.
  - **저관여 공산품 구매자**: 건강과 직결되지 않는 일반 잡화 광고는 우선순위에서 제외한다.
